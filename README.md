<https://salix5.github.io/query/>

## 使用說明
卡名、效果等欄位支援萬用字元 \_ , %  
\_： 任何單一字元  
%： 任意長度的字串  
搜尋 \_ , % 請改成輸入 `$_`, `$%`

Ex.  
機巧（きこう）：  
卡名搜尋`機巧_-%`

可以放置魔力計數器的卡：  
效果搜尋`%給此卡放置%魔力計數器%`
