## 使用說明
卡名、效果等欄位支援萬用字元 \_ , %  
\_： 任何單一字元  
%： 任意長度的字元  
搜尋 \_ , % 請改成輸入 \$\_ , \$%

Ex. 機巧（きこう）：搜尋`機巧_-%`
