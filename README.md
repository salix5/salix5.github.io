## 使用說明

### 卡名欄位支援完整的日文卡名  
日文卡名：  
英文與數字輸入全形、半形皆可  

中文卡名：  
更動部分卡名以反映字段  
找不到請改用關鍵字搜尋


### 文字欄位支援萬用字元 \_ , %  
\_： 任何單一字元  
%： 任意長度的字串  
搜尋 \_ , % 請改成輸入 `$_`, `$%`

Ex.  
機巧（きこう）：  
卡名搜尋`機巧_-%`

可以放置魔力計數器的卡：  
效果搜尋`%給此卡放置%魔力計數器%`

### 類型支援AND、OR、排除特定類型
使用上述條件可以快速搜尋複合類型的卡

Ex.  
協調同步怪獸：  
同步 AND 協調  

放在牌組的卡：  
排除：融合、同步、超量、連結
